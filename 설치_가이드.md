# ⚙️ 슬랙-구글 시트 통합 시스템 설치 가이드 (v1.0)

이 가이드는 프로그램을 처음 전달받은 비전공자 분들도 직접 설정을 완료할 수 있도록 작성되었습니다.

---

## 1. 사전 준비물
1.  **Node.js 설치**: [https://nodejs.org/](https://nodejs.org/) (LTS 버전 추천)
2.  **ngrok 설치**: [https://ngrok.com/](https://ngrok.com/) (슬랙 메시지를 서버로 받기 위해 필수입니다)

---

## 2. API 키 발급받기 (중요!)

### ① 구글 서비스 계정 JSON 키 발급
1.  [Google Cloud Console](https://console.cloud.google.com/) 접속
2.  프로젝트 선택 (또는 새 프로젝트 생성)
3.  **[API 및 서비스] > [사용 설정된 API 및 서비스]**에서 `Google Sheets API`를 찾아 '사용' 클릭
4.  **[IAM 및 관리자] > [서비스 계정]** 클릭
5.  **[서비스 계정 만들기]** 클릭 (이름 작성 후 완료)
6.  생성된 계정의 **[관리] > [키]** 탭 클릭
7.  **[키 추가] > [새 키 만들기] > [JSON]** 선택하여 다운로드 (파일 내용 전체를 복사해 두세요)
8.  **이메일 주소 복사**: 서비스 계정 목록에 있는 이메일 주소를 사용하려는 구글 시트의 **[공유]** 버튼을 눌러 '편집자'로 추가해 주세요.

### ② Slack API 설정
1.  [Slack API](https://api.slack.com/apps) 접속 후 **[Create New App]** 클릭
2.  **[Settings] > [Basic Information]**에서 `Signing Secret` 복사
3.  **[Features] > [OAuth & Permissions]**에서 `Bot User OAuth Token` 생성 및 복사
4.  **권한 설정 (Scopes)**: `app_mentions:read`, `chat:write`, `groups:history`, `im:history`, `incoming-webhook`, `mpim:history`, `channels:history` 추가
5.  **Request URL 설정**: ngrok 실행 후 나온 `https://...` 주소 끝에 `/slack/events`를 붙여 **Event Subscriptions**와 **Interactivity**에 입력

### ③ Gemini API 키 발급
1.  [Google AI Studio](https://aistudio.google.com/app/apikey) 접속
2.  **[Get API key]** 버튼을 눌러 키 생성 및 복사

---

## 3. 프로그램 설치 및 실행
1.  받으신 폴더 안의 `install.bat` 파일을 실행하여 라이브러리를 설치합니다.
2.  `start.bat` 파일을 실행하여 서버를 켭니다.
3.  브라우저에서 `http://localhost:3000`에 접속합니다.
4.  **2단계에서 발급받은 모든 키(1~4번 칸)**를 빈칸에 알맞게 입력하세요.
    - JSON 키는 파일 내용 전체( { ... } )를 그대로 붙여넣으시면 됩니다.
5.  **[모든 설정 저장 및 가동]** 버튼을 누르면 끝입니다!
